## Installation

Install my-project with npm

1. First clone it form the repo

```bash
  git clone https://github.com/Toufiquer/electron-vite-react-chadcn.git
```

/_ Change Directory [for windows] _/

```bash
  cd .\electron-vite-react-chadcn\
```

/_ yarn _/

```bash
  yarn install
```

```bash
  yarn dev
```

/_ npm _/

```bash
  npm install
```

```bash
  npm run dev
```
